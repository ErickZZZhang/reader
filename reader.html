☁ 云同步设置

通过 GitHub Gist 在多台设备间同步阅读进度（章节 + 滚动位置）。字体、主题、宽度等偏好设置不同步，每台设备各自保留。Token 只保存在本机浏览器中，不会上传到页面代码。

手动模式适合在公司网络或流量敏感场景，避免频繁访问 GitHub。可随时点击工具栏的「↗ 去最新」或此处按钮同步。

首次使用：
1. 在 github.com/settings/tokens 创建 Fine-grained token，Account permissions → Gists: Read and write（或用 classic token 勾选 gist）。
2. 粘贴 token 到上方，Gist ID 可留空 —— 点击「保存并同步」会自动创建一个私密 gist。
3. 在其他电脑上，粘贴同一个 token 和上面的 Gist ID 即可。
